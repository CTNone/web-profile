# Test case T001 - Khởi tạo dự án Vite + React

## Mục tiêu
Xác nhận dự án Vite + React với TypeScript được khởi tạo đúng và có cấu trúc cơ bản.

## Test cases
- [ ] Kiểm tra sự tồn tại của `package.json` với dependency `react` và `react-dom`.
- [ ] Kiểm tra `tsconfig.json` có cấu hình TypeScript.
- [ ] Kiểm tra `vite.config.ts` tồn tại và không lỗi khi chạy `npm run build`.
- [ ] Kiểm tra `src/App.tsx` và `src/main.tsx` tồn tại.
